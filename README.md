# ImperiumAPI

API de jogo de estratégia onde jogadores controlam impérios, coletam recursos, guerreiam e negociam.

## Tecnologias
- NestJS
- PostgreSQL
- Prisma
- JWT Auth

## Como rodar
```bash
git clone https://github.com/seu-usuario/imperium-api.git
cd imperium-api
pnpm install
pnpm run start:dev
